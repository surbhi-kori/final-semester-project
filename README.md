# final-semester-project
BCA 6th Semester Project: E- Commerce website
Objectives of the Project
The main objectives of the Vastraa project are:
* To develop a dynamic web-based platform for selling clothing and apparel products.
* To provide an easy-to-use interface that allows users to browse clothing products conveniently.
* To create an admin panel that allows administrators to manage products and website content.
* To store product information securely using a MySQL database.
* To reduce manual effort in updating product information by providing automated management tools.
* To design a responsive website that works on multiple devices such as desktops, tablets, and smartphones.
* To demonstrate the practical use of modern web technologies in building an e-commerce platform.
